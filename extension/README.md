# Chrome 插件版网盘下载器

## 安装

1. 打开 Chrome。
2. 地址栏输入 `chrome://extensions/`。
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择项目里的 `extension` 目录。

```text
netdisk-paste-downloader/extension
```

## 使用

1. 确认当前 Chrome 已登录百度网盘或夸克网盘。
2. 点击浏览器右上角的插件图标。
3. 粘贴一段或多段分享文字：

```text
通过网盘分享的文件：01.mp4
链接: https://pan.baidu.com/s/1hITBASN1CM7eu7YdfVpO1Q 提取码: xibc
```

4. 点击“开始”。

插件会自动识别链接和提取码。如果粘贴了多条分享，会分别打开多个标签页。进入网盘页面后，右下角会显示“网盘插件”状态提示，并尝试输入提取码、点击提取、全选、下载。

更新插件代码后，需要在 `chrome://extensions/` 里点击本插件的“重新加载”按钮。

## 注意

插件不会读取或导出 cookies，也不会绕过验证码、登录、风控、会员限制或平台权限。遇到确认弹窗、验证码、客户端下载提示时，需要在页面里手动处理。
